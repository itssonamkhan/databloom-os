- TypeScript
- Tailwind CSS
- React
- GitHub
- Vercel

## 🚀 About

DataBloom OS is a personal project built to make learning Data Analytics enjoyable through gamification, progress tracking, and an aesthetic, Pinterest-inspired interface.

This project is actively under development, with upcoming modules for SQL, Python, Power BI, Statistics, Interview Prep, and AI-powered learning.
