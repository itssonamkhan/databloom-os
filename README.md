# ✨ About

DataBloom OS is a modern web application designed to make learning Data Analytics enjoyable through gamification, progress tracking, AI-powered assistance, and a cozy Pinterest-inspired interface.

Instead of learning from scattered notes and disconnected resources, DataBloom OS provides a single platform where learners can study, practice, revise, and track their complete learning journey.

The project is being independently designed and developed by **Sonam Khan** using modern web technologies.
